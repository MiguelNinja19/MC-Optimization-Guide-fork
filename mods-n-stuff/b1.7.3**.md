## Babric
| Mod | Notes | Client/Server |
|:---:|:---:|:---:|
| [SmoothBeta](https://github.com/mineLdiver/smoothbeta) | Multidraw terrain renderer,fastutil,IdentityHashMap,Custom entity registry. | Both |
| [Gambac](https://modrinth.com/mod/gambac) | deAWTs the game and fixes colors being inverted on Apple Silicon chips. | Client |
| [Entity Culling](https://modrinth.com/mod/entityculling) | - | Client |
| [Font Fixer](https://github.com/ThexXTURBOXx/BlockHelper) | Fixes the Font Renderer. | Client |
| [UniTweaks](https://modrinth.com/mod/unitweaks) | All-in-one fix and tweak mod. | Client |
| [RetroAuth](https://github.com/matthewperiut/retroauth) | Fixes skins and authentication. | Both |
| [UniTweaksTelsAddons](https://github.com/telvarost/UniTweaksTelsAddons-StationAPI) | Provides features from incompatible mods of mine to UniTweaks. | Both |

### Java Notes:
Babric runs on Java 17 and above. Java 25 works on babric, but you have to add `-Djava.locale.providers=JRE` to your java arguments to fix some minor issues with dates.

### Shaders Notes:
You can use shaders by using [Macula](https://github.com/mineLdiver/Macula) but it's incompatible with [SmoothBeta](https://github.com/mineLdiver/smoothbeta) (it causes some colors,etc. issues)
